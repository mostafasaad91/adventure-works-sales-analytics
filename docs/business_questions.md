# AdventureWorks2022 — أسئلة البيزنس للمشروع

## حقائق أساسية عن الداتا
- الفترة: مايو 2011 → يونيو 2014 (3 سنين)
- الأوردرات: 31,465 أوردر
- الإيراد الكلي: $123.2M
- العملاء: 19,119 عميل
- القنوات: 88% Online / 12% Store (SalesPerson)

---

## ⭐ السؤال الرئيسي للمشروع

> **"إيرادات $123M في 3 سنين — فين الفلوس بتيجي، فين بتتسرب (خصومات / منتجات خاسرة / عملاء بيهربوا)، وإيه أكبر 3 فرص نمو؟"**

---

## 1. المبيعات والنمو 📈
- [x] إزاي تطور الإيراد شهرياً وسنوياً؟ فين الـ seasonality (شهور الذروة والهبوط)؟
- [x] الـ YoY growth بيتسارع ولا بيتباطأ؟
- [x] النمو جاي من عملاء جدد ولا زيادة إنفاق العملاء الحاليين؟
- [x] متوسط قيمة الأوردر (AOV) بيتغير مع الوقت إزاي؟ Online vs Store؟

## 2. المنتجات والربحية 💰
- [x] إيه أعلى المنتجات/الفئات **ربحاً** مش بس إيراداً؟
- [x] الخصومات (SpecialOffer) بتزود الكمية فعلاً ولا بتاكل الهامش؟
- [x] إيه المنتجات اللي بتتباع مع بعض؟ (فرص cross-selling)
- [x] منتجات راكدة: مخزون عالي + مبيعات ضعيفة = فلوس محبوسة فين؟
- [x] إيه أكتر المنتجات اللي بترجع أو بتتعطل؟ (scrap/returns)

## 3. العملاء 👥
- [x] مين أفضل 20% عملاء وبيمثلوا كام % من الإيراد؟ (Pareto)
- [x] تقسيم RFM: مين الـ VIP، مين معرض للـ churn، مين نايم يستاهل إعادة تنشيط؟
- [x] الـ retention: العميل الجديد بيرجع يشتري خلال قد إيه؟ كام % بيختفوا بعد أول أوردر؟
- [x] سلوك عميل الـ Online مختلف عن عميل الـ Store إزاي؟

## 4. الجغرافيا والقنوات 🌍
- [x] إيه أقوى وأضعف الـ Territories؟ الضعف في عدد العملاء ولا قيمة الأوردر؟
- [x] فيه مناطق نموها عالي بس حجمها صغير = فرصة استثمار؟
- [x] الشحن: متوسط الوقت بين OrderDate و ShipDate — فيه مناطق بتتأخر؟

## 5. فريق المبيعات 🎯
- [x] مين بيحقق الـ Quota ومين لا؟ هل الـ Quota عادلة بين الـ Territories؟
- [x] الـ Commission بتترجم لأداء فعلاً؟
- [x] مبيعات SalesPerson (B2B) مقابل Online — أنهي قناة تستاهل استثمار أكتر؟
